# Console-Based User & Admin Management System

## Overview
The Console-Based User & Admin Management System is a Java application developed to manage users through a command-line interface. The system provides separate functionalities for administrators and users, ensuring secure access and efficient management of user records. Database operations are performed using JDBC and MySQL.

## Features

### User Module
- User Registration
- User Login
- View Profile
- Update Profile
- Delete Account

### Admin Module
- Admin Login
- View All Users
- Add New Users
- Update User Details
- Delete User Accounts
- Manage User Records

### Database Operations
- Create Records
- Read Records
- Update Records
- Delete Records
- Secure Queries using PreparedStatement

## Technologies Used
- Java
- JDBC
- MySQL
- SQL
- Object-Oriented Programming (OOP)

## Project Objectives
- Implement role-based access control for Admin and User.
- Perform CRUD operations using JDBC and MySQL.
- Maintain secure database interactions using PreparedStatement.
- Demonstrate practical implementation of Java and database connectivity.

## System Workflow
1. User/Admin selects an option from the main menu.
2. Credentials are validated using the database.
3. Based on the role, corresponding functionalities are displayed.
4. CRUD operations are performed on the database.
5. Results are displayed through the console interface.

## Database Connectivity
The application uses JDBC to establish a connection with MySQL and execute SQL queries securely.
